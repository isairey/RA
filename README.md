# 🧠 Realidad Aumentada con HTML

## 📌 Descripción

Este proyecto muestra una implementación básica de **Realidad Aumentada (RA)** utilizando tecnologías web como **HTML, JavaScript y librerías de AR**.

Permite visualizar objetos 3D o contenido digital sobre el entorno real usando la cámara del dispositivo.

---

## 🚀 Características

- 📷 Uso de la cámara del dispositivo  
- 🧊 Visualización de objetos en RA  
- 🌐 Funciona directamente desde el navegador  
- ⚡ Ligero y fácil de usar  

---

## 🛠 Tecnologías

- HTML5  
- JavaScript  
- A-Frame / AR.js
